Giới thiệu về mảng một chiều

Mảng một chiều một cấu trúc dữ liệu đầu tiên được dạy khi mới bắt đầu học lập trình, nên có lẽ không còn xa lạ gì nữa. Mảng một chiều gồm một tập hợp dãy các phần tử liên tiếp nhau trên không gian bộ nhớ. Phần tử đầu tiên của mảng có vị trí bắt đầu ở vị trí 0. Mảng một chiều cho phép truy cập vào một phần tử trong mảng tại vị trí bất kỳ với tốc độ là hằng số.

Mảng một chiều

Mảng tĩnh là mảng một chiều có độ dài là hằng số. Khi ta cấp phát mảng tĩnh với độ dài bất kỳ, độ dài đó sẽ không thay đổi trong xuyên suốt vòng đời của nó và nó sẽ được tự động giải phóng bởi chương trình.
Mảng động là mảng một chiều được cấp phát với dung lượng thuộc vùng nhớ Heap. Mảng động cho phép ta có thể thu hồi dữ liệu, nới rộng và thu nhỏ độ dài tùy ý, luôn được ưu tiên sử dụng hơn vì nó có thể co giãn để chứa dữ liệu.
- Thư viện Array List được xây dựng, sử dụng tương tự như mảng một chiều nhưng có nhiều chức năng hơn, dễ sử dụng hơn và đặc biệt là tự quản lý bộ nhớ, tự thu hồi khi không cần sử dụng. Khuyên thường dùng với những tập dữ liệu nhỏ hoặc vừa đủ lớn. Array List cũng hỗ trợ đa dạng tính năng, phù hợp với các kiểu dữ liệu built-in, hỗ trợ Iterator và tương tác với một số lớp trong C++.

Hướng dẫn sử dụng cơ bản Array List

Bài viết này sẽ giới thiệu chính về lớp Array List, bên trong nó được cài đặt bằng mảng một chiều, vì thế có thể sử dụng nó như mảng một chiều bình thường.
Cú pháp khai báo Array List : arraylist<typename>. Trong đó typename là loại dữ liệu cần lưu trữ.
Bạn có thể khởi tạo Array List rỗng, từ danh sách cho trước, hoặc sao chép dữ liệu từ một Array List khác.

void main() { //Khởi tạo Array List rỗng arraylist<int> list; std::cout << list; //Khởi tạo Array List có sẵn các phần tử arraylist<int> list2{1, 2, 3}; std::cout << list2; arraylist<char> list3{'a', 'b', 'c'}; std::cout << list3; //Khởi tạo sao chép từ list2 arraylist<int> list4(list2); std::cout << list4; arraylist<int> list5 = arraylist<int>{1, 3, 5}; std::cout << list5; }

Các phương thức thêm, xóa, sửa dữ liệu cơ bản : pushFront, pushBack, pushAt, popFront, popBack, popAt.
Các phương thức trên dùng để thêm, xóa, sửa một phần tử trong Array List.

void main() { arraylist<int> list{5, 2, 1, 3, 7}; //Xóa một phần tử list.popBack(); list.popFront(); list.popAt(1); //Thêm một phần tử list.pushBack(10); list.pushFront(12); list.pushAt(31, 2); //Gán lại một phần tử list[2] = 15; std::cout << list; }

Các phương thức phục vụ công việc tìm kiếm, sắp xếp, đảo mảng cơ bản : indexOf, sort, reverse

void main() { arraylist<int> list{2, 5, -6, 0, 3, 7, 1}; //Tìm kiếm std::cout << "So -6 o vi tri " << list.indexOf(-6) << std::endl; //Sắp xếp tăng dần list.sort(); std::cout << list; //Đảo mảng list.reverse(); std::cout << list; }

Ngoài ra Array List còn hỗ trợ lớp Iterator dành cho việc duyệt và thao tác với nó.
Iterator này gồm có các phương thức tăng, giảm ( ++, -- ) như con trỏ, có thể dùng nó thực hiện một số giải thuật như tìm kiếm nhị phân, Quick Sort,... . Sử dụng Iterator bằng cách gọi các phương thức của Array List như : begin, end, rbegin, rend, cbegin, cend, crbegin, crend.

void main() { arraylist<int> list{2, 5, -6, 0, 3, 7, 1}; auto iter = list.begin() + 3; std::cout << "Phan tu thu 3 : " << *i; //Xuất mảng dùng vòng lặp for(int i : list) std::cout << i << ' '; std::cout << std::endl; //Tạo list2 gồm các phần tử trong nửa đoạn [1, 4) của list arraylist<int> list2(list.begin() + 1, list.begin() + 4); std::cout >> list2; }

Như vậy các bạn đã có thể sử dụng tốt lớp Array List mà không nhọc nhằn gì nhiều. Để hiểu rõ hơn nữa thì hãy qua phần bài tập để nắm rõ hơn nhé. Hi vọng rằng nó hữu ích với các bạn.

Bài tập vận dụng

Bài 1. Cho hai dãy số nguyên tăng dần, hãy gộp hai dãy đó thành một dãy mới cũng tăng dần.

void main() { arraylist<int> list1{-10, -2, 0, 6, 11}, list2{3, 7, 8, 14, 22}; //Cách 1 : gộp hai mảng và gọi hàm sắp xếp arraylist<int> list3 = list1 + list2; list3.sort(); std::cout << list3; //Cách 2: so sánh từng phần tử và thêm vào arraylist<int> list4; auto i = list1.begin(), j = list2.begin(); while (i != list1.end() && j != list2.end()) if (*i <= *j) { list4.pushBack(*i); ++i; } else { list4.pushBack(*j); ++j; } list4 += arraylist<int>(i, list1.end()) + arraylist<int>(j, list2.end()); std::cout << list4; }
